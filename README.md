# 📊 Análises de Dados - Amazon Sales

## 🎯 Objetivo
Este repositório contém análises exploratórias realizadas com base em dados de vendas da Amazon, com foco em descontos, categorias e comportamento de vendas.

## 🛠️ Ferramentas Utilizadas
- Python
- Pandas
- Jupyter Notebook
- Git e GitHub

## 📁 Estrutura do Projeto
- `data/` → Base de dados utilizada
- `notebooks/` → Análises e estudos realizados
- `requirements.txt` → Bibliotecas necessárias

## 📈 Principais Análises
- Análise de descontos médios
- Comparação de vendas por categoria
- Exploração estatística dos dados
